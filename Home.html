Welcome to the Betting Web Application Hackathon

Introduction

Are you ready to revolutionize the world of online betting? Join us in this hackathon challenge to develop a cutting-edge betting web application using MERN stack or Django framework. Your task is to create a platform where users can engage in thrilling betting experiences while competing against each other in various team-based competitions.

Objectives and Key Features

  • Develop a robust web application that allows users to:
  • Sign up or log in securely to the platform.
  • View and select from a list of ongoing team-based competitions.
  • Place bets on their chosen teams with a minimum bet of Rs 50.
  • Accrue winnings based on their bet amount and the outcome of the competition.
  • Create a betting pool where users' bets contribute to a collective pot.
  • Implement real-time updates to reflect changes in betting odds and competition status.
  • Design a user-friendly profile page displaying users' betting history, wins, losses, and remaining balance.
  • Ensure secure payment transactions and user data protection.
  • Deploy the application on a cloud platform for accessibility.
